# 📂 Kho Bài Tập Thực Hành OOP & Thuật Toán

Đây là nhánh lưu trữ các bài tập code hàng ngày (Daily Coding), bài tập Lab trên lớp và các bài giải thuật toán mà tôi thực hiện để rèn luyện tư duy lập trình C++.

## 🛠️ Danh sách bài tập (Exercises List)

### 🏛️ Lập trình Hướng đối tượng (OOP)
Các bài tập rèn luyện 4 tính chất: Đóng gói, Kế thừa, Đa hình, Trừu tượng.

| Tên bài tập | Kỹ thuật chính | Trạng thái |
| :--- | :--- | :---: |
| **Quản lý Sinh viên** | Nạp chồng toán tử (`>>`, `<<`), `static` variable, Chuẩn hóa chuỗi. | ✅ |
| **Quản lý Thư viện** | Đa hình (`virtual function`), Lớp trừu tượng, Quản lý bộ nhớ (`vector<Pointer>`). | ✅ |
| **Class Phân số** | Nạp chồng toán tử cộng trừ nhân chia, rút gọn phân số. | ✅ |

## 📝 Ghi chú quan trọng
* **Môi trường Code:** Visual Studio 2026.
* **Tiêu chuẩn:** Code tuân thủ quy tắc đặt tên biến rõ ràng, có comment giải thích logic.
* **Fix lỗi thường gặp:**
    * Xử lý trôi lệnh (`cin.ignore()`) khi nhập chuỗi sau số.
    * Quản lý bộ nhớ Heap (luôn `delete` sau khi `new`).
    * Tránh xung đột thư viện bằng cách hạn chế `using namespace` bừa bãi trong header.
